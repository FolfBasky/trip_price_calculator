# Contributing

Спасибо за интерес к проекту.

## Как внести изменения

1. Сделайте fork репозитория.
2. Создайте отдельную ветку.
3. Запустите проект локально и проверьте изменения.
4. Перед pull request проверьте Python и JavaScript:

```bash
python -m py_compile backend/*.py
node --check frontend/js/*.js
```

## Стиль коммитов

Пишите коротко и понятно:

```text
fix: correct pdf route rendering
feat: add trip history
chore: update readme
```
